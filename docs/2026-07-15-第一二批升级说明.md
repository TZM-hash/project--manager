# 第一～二批升級交付說明

日期：2026-07-15  
範圍：UI 基礎與繁體術語、資料工作台與個人檢視

## 第一批：UI 基礎與繁體術語

- 界面固定文案以繁體中文為規範來源，保留簡體顯示轉換能力。
- 側邊導覽分為「業務工作／報表分析／系統管理」，折疊狀態支援 Tooltip、鍵盤焦點與 ARIA。
- 狀態與風險顏色改為表達業務語意，不再單純依百分比使用危險色。
- 高頻列表導入共享空狀態，說明無資料原因並提供下一步操作。
- 前端 CSS 拆分為基礎、元件、頁面、主題四層。
- `site.js` 改為按頁面標記載入 ES Modules，避免所有頁面初始化全部元件。

## 第二批：資料工作台與個人檢視

- 新增 `SavedDataViews` 資料表、EF Core Migration、白名單定義與個人檢視服務。
- 專案總覽、未結案報表、我的專案、保養訂單支援：
  - 系統預設檢視；
  - 儲存個人篩選、可見欄位與行距；
  - 刪除個人檢視；
  - 設定頁面預設檢視；
  - 跨瀏覽器保存個人設定。
- 未結案報表使用 17 個穩定欄位鍵，預設只顯示九個高優先欄位。
- 專案工號與專案名稱在桌面橫向捲動時保持凍結；長說明使用兩行截斷。
- 列管理至少保留一個可設定欄位，固定識別欄與操作欄不可隱藏。
- 批量刪除、Excel 匯入與匯出增加處理中狀態及完成摘要。
- 新增鍵盤操作、720px 有效寬度（近似 200% 縮放）與顏色對比視覺測試。
